# ds-free-api：DeepSeek API 代理

> **仓库**：https://github.com/NIyueeE/ds-free-api (535⭐, Rust)  
> **核心**：DeepSeek 网页端 API 代理（OpenAI+Anthropic 兼容）  
> **Hermes_Rust_Operit_App 评分**：★★（用官方 API 更稳定）

---

## 核心能力

- DeepSeek 网页端→OpenAI 兼容接口转换
- 支持流式输出
- Anthropic 兼容接口

## 对 Hermes 的作用

Hermes 的 provider.rs 已支持 DeepSeek 官方 API（通过 openai 兼容模式）。ds-free-api 的价值在于**免费**（通过网页端），但稳定性和速率不如官方 API。
