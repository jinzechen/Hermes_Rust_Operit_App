六、Agent 引擎核心架构

┌──────────────────────────────────────────────────────────────┐ │ AgentManager │ │ ┌──────────────────────────────────────────────────────┐ │ │ │ agent_loop(session_id) { │ │ │ │ loop (max 20 iterations) { │ │ │ │ 1. build_messages(session_id) │ │ │ │ 2. provider.chat_completion(messages, tools) │ │ │ │ 3. if no tool_calls → return final_content │ │ │ │ 4. for each tool_call: │ │ │ │ result = execute_tool_call(tc) │ │ │ │ append tool_message to session │ │ │ │ } │ │ │ │ } │ │ │ └──────────────────────────────────────────────────────┘ │ │ │ │ Dependencies: │ │ ├─ ToolRegistry (77 lines, HashMap-based) │ │ ├─ LlmProvider trait (Generic + Anthropic) │ │ ├─ SessionManager (HashMap>) │ │ └─ AppConfig (model, api_key, endpoint, temp, max_tokens) │ └──────────────────────────────────────────────────────────────┘
